# Aura — personal study platform

A local-first study tracker for a real university semester: per-subject weekly review progress, past-paper practice, a focus timer that logs real study time, and an AI end-of-day reflection.

> Work in progress — full README with screenshots lands at the polish milestone.

## Run

```bash
npm install
cp .env.example .env.local   # optional: add ANTHROPIC_API_KEY for reflections
npm run dev
```
